##Rodar código em C

### No Linux
```
gcc nomedoarquivo.c -o nomedoarquivo && ./nomedoarquivo
```
### Outro método no Linux
```
gcc nomedoarquivo.c && ./a.out
```
### No Windows
```
gcc nomedoarquivo.c -o nomedoarquivo.exe 
./nomedoarquivo.exe
```
